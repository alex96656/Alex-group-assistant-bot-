# 🤖 Lexxie MiaBot

A powerful Telegram assistant bot built with **TypeScript** and the **Grammy Framework**.

Lexxie MiaBot provides moderation tools, utility commands, user management, welcome systems, and more.

## ✨ Features

✅ Telegram Assistant  
✅ Moderation System  
✅ Ban / Kick / Mute / Warn System  
✅ Admin Management  
✅ Welcome Messages  
✅ Anti-Link Protection  
✅ Notes System  
✅ User Profile System  
✅ Dashboard  
✅ Database Backup  
✅ Fun Commands  
✅ AI Features (Coming Soon)

## 📌 Commands

### 👤 General
/start - Start the bot  
/help - View commands  
/botinfo - Bot information  
/ping - Check response speed  
/profile - View profile  
/id - Get Telegram ID  

### 🛡️ Moderation
/ban - Ban a user  
/kick - Remove a user  
/mute - Mute a user  
/unmute - Unmute a user  
/warn - Warn a user  
/warnings - View warnings  
/promote - Promote admin  
/demote - Remove admin

### ⚙️ Utilities
/users - View bot users  
/admins - View group admins  
/stats - Bot statistics  
/backup - Database backup  
/server - Server information

## 🛠️ Built With

- TypeScript
- Node.js
- Grammy Framework
- Telegram Bot API
